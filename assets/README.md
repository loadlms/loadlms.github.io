Coloque aqui o arquivo do ícone (favicon) em PNG.

- Nome recomendado: `favicon.png`
- Tamanho sugerido: 180x180 (mas pode ser qualquer, o navegador faz o downscale)
- Depois do upload, o `index.html` já referencia `assets/favicon.png?v=20251111`

Se estiver usando GitHub Pages com a pasta `docs`, garanta que esta pasta `assets` esteja dentro de `docs/` também.
